# yewon

이 디렉토리는 운영 화면과 장애 분석 관련 시각화 자료, 목업, 실험용 HTML 파일을 모아 둔 작업 공간입니다. 배포 대상 애플리케이션 코드라기보다 아이디어와 UI 시안을 빠르게 확인하는 용도에 가깝습니다.

## 이 디렉토리가 담는 내용

- 운영 대시보드 목업
- incident-agent 시각화 샘플
- 보조 문서와 실험용 HTML 파일

## 디렉토리 구조

```text
yewon/
├── docs/                   시각화 문서와 HTML 샘플
├── ops-dashboard-mockup.html
└── README.md
```

## 참고

- 실제 운영 API는 `incident-api/`
- 실제 프론트 구현은 `frontend/`
- 이 디렉토리는 아이디어 검토용 자료로 보는 편이 맞습니다.
