# msa-docs

이 디렉토리는 MSA 전환 과정에서 정리한 설계 문서와 작업 기록을 모아 두는 공간입니다. 현재 실행 코드 자체보다 아키텍처 의도와 변경 이력을 이해할 때 유용합니다.

## 이 디렉토리가 담는 내용

- 서비스 맵과 역할 정리
- 배포 준비 메모
- 마이그레이션 기록
- 큐 서비스 리팩터링 관련 자료

## 디렉토리 구조

```text
msa-docs/
├── queue-service/
│   └── assets/             큐 서비스 관련 이미지 자산
└── README.md
```

## 구조를 읽는 방법

- 전체 MSA 흐름은 최상위 문서들
- 특정 큐 서비스 논의는 `queue-service/`
- 현재 코드와 문서가 다를 수 있으니 최종 판단은 소스 기준으로 해야 합니다.
