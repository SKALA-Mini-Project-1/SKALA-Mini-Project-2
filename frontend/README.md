# Ticketing Frontend (Vue 3 + TypeScript + Vite)

## Getting Started

1. Install dependencies: `npm install`
2. (Optional) Override per-service proxy targets:
   - `VITE_USER_AUTH_API_URL` default `http://localhost:18081`
   - `VITE_CONCERT_API_URL` default `http://localhost:18082`
   - `VITE_QUEUE_API_URL` default `http://localhost:18083`
   - `VITE_TICKETING_API_URL` default `http://localhost:18084`
   - `VITE_PAYMENT_API_URL` default `http://localhost:18085`
3. Start dev server: `npm run dev`
4. Build for production: `npm run build`
